# Adivinar un Número

## Comprehensive Documentation

### Features
- User-friendly interface
- Flexible guessing range
- Tracks number of attempts

### Usage Guide
1. Run the program.
2. Choose a range for the number to be guessed (e.g., 1 to 100).
3. Input your guesses until you find the correct number.

### Code Explanation
The program utilizes a simple algorithm to randomly select a number within the specified range and compares it to user guesses, providing feedback on whether the guess is too high, too low, or correct.

### Examples
- If the number is 56 and the user guesses 50, the output will be "Too low!".
- If the user guesses 60, the output will be "Too high!".
- Upon guessing 56, the output will be "Congratulations! You guessed it!".
